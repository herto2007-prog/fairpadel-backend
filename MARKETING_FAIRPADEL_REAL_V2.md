## 💬 RESPUESTAS TIPO PARA COMENTARIOS

### Cuando preguntan "¿Es gratis de verdad?"
```
¡Sí! El americano es gratis sin letra chica. No te pedimos tarjeta, no hay límite de uso, nada.

Para los torneos normales (eliminación directa), el organizador paga una comisión por inscripto.

Pero vos como jugador no pagás nunca. Y el americano es gratis para todos. 😊
```

### Cuando preguntan "¿Cómo funciona?"
```
¡Hola! Es una web/app. Entrás, creás un torneo americano gratis, pasás el link a tus amigos, y ellos se inscriben solos.

El sistema arma las rondas automático y lleva el puntaje.

¿Querés que te pase el link? 📲
```

### Cuando preguntan "¿Y si no tengo grupo de pádel?"
```
¡Buena pregunta! Por ahora FairPadel sirve para organizar torneos con gente que ya conocés (amigos, compañeros de trabajo, etc).

Estamos pensando en cómo conectar jugadores sueltos, pero todavía no está listo.

¿Tenés un grupo con quien jugar? Ahí es donde brilla la app. 🎾
```

### Cuando preguntan "¿Para qué sirve como organizador?"
```
¡Hola! Como organizador podés:

✅ Crear torneos con fixture automático
✅ Recibir inscripciones online
✅ Cobrar inscripciones (transferencia o Bancard)
✅ Enviar notificaciones WhatsApp a jugadores
✅ Cargar resultados y que el bracket avance solo
✅ Ver rankings y estadísticas

¿Te interesa que te hagamos una demo? Escribinos al WhatsApp del link en bio. 📲
```
